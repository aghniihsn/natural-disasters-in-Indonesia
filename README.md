# Dashboard Bencana Alam Sumatera 2025

Proyek ini adalah dashboard interaktif berbasis web yang menampilkan data terkini bencana banjir dan longsor di Sumatera pada tahun 2025. Dashboard ini dibuat sebagai tugas untuk mata kuliah Sistem Informasi Geografi.

## Fitur Utama

- **Statistik Terkini**: Menampilkan jumlah korban tewas, hilang, dan pengungsi berdasarkan data BNPB/Detikcom.
- **Sebaran Korban**: Distribusi korban jiwa di Aceh, Sumatera Utara, dan Sumatera Barat.
- **Dampak Infrastruktur**: Informasi kerusakan jembatan, fasilitas kesehatan, sekolah, rumah ibadah, dan jumlah korban luka.
- **Analisis Spasial**: Peta interaktif menggunakan Leaflet yang menampilkan:
  - Area deforestasi (hulu) sebagai penyebab utama banjir.
  - Titik-titik lokasi bencana terparah.
  - Analisis citra satelit dan penjelasan singkat dampak kerusakan hutan.

## Teknologi yang Digunakan

- **HTML, CSS (Tailwind, custom style)**
- **JavaScript**
- **Leaflet.js** untuk peta interaktif
- **Font Awesome** untuk ikon

## Cara Menggunakan

1. Buka file `index.html` di browser.
2. Lihat statistik dan informasi bencana pada dashboard utama.
3. Klik tombol "ANALISIS SPASIAL: LIHAT PETA PENYEBAB" untuk membuka peta interaktif.
4. Gunakan tombol "Kembali" untuk kembali ke dashboard.

## Sumber Data
